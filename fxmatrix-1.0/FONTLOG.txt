FONTLOG for the FX Matrix fonts

This file provides detailed information on the FX Matrix Font Software.
This information should be distributed along with the FX Matrix fonts
and any derivative works.

Basic Font Information

FX Matrix is a typeface inspired by the output of Epson FX 9-pin dot
matrix printers from the 80's. Its goal is to closely resemble the
output of the EPSON FX-80 printer as supporting the various print
modes originally by the printer.
